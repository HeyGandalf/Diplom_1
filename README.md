## Дипломный проект. Задание 1: Юнит-тесты

### Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие классы `Bun`, `Burger`, `Ingredient`, `Database`.

Процент покрытия — 100% (отчёт: `htmlcov/index.html`).

### Структура проекта

- `praktikum` — пакет, содержащий основной код программы
- `tests` — пакет с автотестами, разделёнными по классам (`test_bun.py`, `test_burger.py` и т.д.)
- `requirements.txt` — зависимости проекта

### Установка зависимостей

```bash
pip install -r requirements.txt

### Запуск автотестов и создание отчёта о покрытии
